# Tiebreaker

A Claude Code skill that settles hard decisions.

You're stuck between options. Tiebreaker builds a Pugh matrix — rows are your
options, columns are the factors that matter, cells are scores from 1 to 5 —
weighs the factors, ranks the options, and hands you a clear verdict.

## How it works

1. You bring a decision and the options.
2. Tiebreaker asks for the factors that matter and their weights.
3. It fills the 1-5 cells and computes weighted scores.
4. It returns a ranked verdict you can actually trust.
